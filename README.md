# JJ University Study Method Challenge

**전주대학교 학습법 노하우 경진대회 기록 레포지토리** <br/>

---

## Overview <br/>

본 레포지토리는 전주대학교 학습법 노하우 경진대회 참여 과정에서
작성한 학습 기록과 공부 전략을 정리한 공간입니다.

단순한 결과 공유가 아니라,
**실제 학습 과정과 전략, 시행착오까지 포함한 ‘실전형 공부법 아카이브’**를 목표로 합니다.

* 개인 블로그 기반 학습 기록 <br/>
* 주차별 목표 설정 및 실행 과정 기록 <br/>
* 실제 효과 있었던 공부 방법 정리 <br/>

---

## Objective

이 프로젝트의 핵심 목표는 다음과 같습니다.

* 재현 가능한 공부법 구축
* 학습 과정의 구조화 (기록 → 복습 → 개선)
* 개인 학습 전략을 타인에게 공유 가능한 형태로 정리

## Main Study Method

### Cornell Note System (코넬노트)

본 프로젝트의 핵심 학습 전략은
**코넬노트 기반 필기 및 복습 시스템**입니다.

* 수업 내용 구조화
* 키워드 기반 복습
* 요약을 통한 장기 기억 강화

아이패드 + 프로크리에이트를 활용하여
개인 맞춤형 템플릿으로 변형하여 사용합니다.
